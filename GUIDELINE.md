## How to run code

If you use yarn

```sh
yarn
```

Or npm

```sh
npm install
```

Copy file environment

```sh
cp .env.example .env

```

Here are all the variables used in the source code environment.

How to start app with yarn

```sh
yarn dev
```

Or npm

```sh
npm run dev
```
